# xplad [XPLA — Cosmos SDK + EVM + CosmWasm]

`Instance.xplad()` boots an [XPLA](https://xpla.io) node: a Cosmos SDK chain with both EVM and CosmWasm. The native denom (`axpla`) uses 18 decimals, so the instance defaults to larger validator stake/balance than `cosmosBase`.

XPLA publishes an official image, so this instance is **container-first** — see the [Container Runtime guide](/docs/guides/docker). Nothing to install but Docker.

## Usage

```ts
import { Instance } from 'starskiff';

const instance = Instance.xplad({
  accounts: [{ mnemonic: '...', coins: '1000000000000000000000axpla' }],
});
await instance.start(); // pulls + runs ghcr.io/xpladev/xpla

instance.evmUrl; // http://localhost:8545 — EVM JSON-RPC
```

## Runtime

| Call | Runtime |
| ---- | ------- |
| `Instance.xplad()` | Container, `XPLA_DEFAULT_IMAGE` (pinned to the version XPLA mainnet runs) |
| `Instance.xplad({ image: 'my/xpla:tag' })` | Container, your image |
| `Instance.xplad({ binary: 'xplad' })` | Local binary on `PATH` |

## Defaults

| Parameter          | Default            |
| ------------------ | ------------------ |
| `image`            | `XPLA_DEFAULT_IMAGE` (`ghcr.io/xpladev/xpla:v1.10.0`) |
| `binary`           | `"xplad"` (in-image executable; also the PATH name when opted in) |
| `chainId`          | `"dimension_37-1"` |
| `denom`            | `"axpla"`          |
| `prefix`           | `"xpla"`           |
| `validatorBalance` | `1e23` (`100000000000000000000000`) |
| `validatorStake`   | `1e22` (`10000000000000000000000`)  |
| `evmPort`          | `8545`             |

## Precompiles

`XPLA_DEFAULT_PRECOMPILES` mirrors `@xpla/evm`'s `PRECOMPILE_ADDRESSES` (v1.9.0) and is written to `evm.params.active_static_precompiles` by default. The `activeStaticPrecompiles` parameter has three-state semantics:

- **omitted** → XPLA's default set
- **explicit `undefined`** → binary default (genesis untouched)
- **`[]` / `[...]`** → disable all / overwrite

```ts
import { Instance, XPLA_DEFAULT_PRECOMPILES } from 'starskiff';

const instance = Instance.xplad({
  activeStaticPrecompiles: [...XPLA_DEFAULT_PRECOMPILES],
});
```
