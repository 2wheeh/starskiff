# gaiad [Cosmos Hub — IBC counterparty]

`Instance.gaiad()` boots a [Cosmos Hub](https://github.com/cosmos/gaia) node. Gaia ships IBC, which makes it useful as an IBC counterparty chain.

The instance patches Gaia's `feemarket` genesis — `fee_denom` is aligned with the instance `denom`, and gas prices are set near zero so test transactions don't need fee tuning.

## Usage

```ts
import { Instance } from 'starskiff';

const instance = Instance.gaiad({
  chainId: 'cosmoshub-test-1',
  denom: 'uatom',
  accounts: [{ mnemonic: '...', coins: '1000000000uatom' }],
});
await instance.start();
```

## Parameters

`GaiadParameters` extends [`CosmosChainParameters`](/docs/api/instance#parameters) with:

| Parameter | Type     | Default   | Description        |
| --------- | -------- | --------- | ------------------ |
| `binary`  | `string` | `"gaiad"` | Path to the binary |
