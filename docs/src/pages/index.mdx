# Overview

```ts
import { Instance, testAccounts } from 'starskiff';

const [alice] = testAccounts;

const instance = Instance.simd({
  chainId: 'test-1',
  accounts: [{ mnemonic: alice.mnemonic, coins: '1000000000stake', name: alice.name }],
});

await instance.start(); // real simd node, first block in seconds

// instance.rpcUrl  → http://localhost:26657 (CometBFT RPC)
// instance.grpcUrl → http://localhost:9090  (gRPC)
// instance.apiUrl  → http://localhost:1317  (REST)

await instance.stop(); // node killed, temp home dir removed
```

Not a mock — the actual node (from a binary or a chain image), booted with a patched genesis, funded accounts, and health-checked until the first block. Inspired by [prool](https://github.com/wevm/prool) (test instances for Ethereum).
